# How To Use
Elo Plugin For 0.15 Servers

How to use?

You must first add the necessary permission to the groups.
Required perms and sample groups are given below

Example:

Player :
- elo.command
- top.command
- seeelo.command

Admin : 
- elo.command
- top.command
- seeelo.command
- addelo.command
- removeelo.command
